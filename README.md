## Features

This mod adds several new biomes to the game along with new blocks and flowers.

### **New biomes:** 
- Salt Flat
- Shrubland
- Subalpine Meadow
- Mediterranean Scrubland
- Flooded Delta
- Dead Forest
- Temperate Rainforest
- Gravel Flats
- Alpine Forest
- Rocky Fields
- Coral Forest
- Abyssal Trenches
- Mosswood Forest
- Mist Coast
- Rocky Shrubland
- Azalea Forest
